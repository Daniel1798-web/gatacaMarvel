Marvel Characters
This React Native project displays Marvel characters using the Marvel API. The characters are displayed in English.

Installation
To install this project, follow these steps:


Clone this repository to your local machine:
git clone https://github.com/Daniel1798-web/gatacaMarvel/new/master

Install the required dependencies using npm or yarn:
npm install

Start the development server:
npm start


Usage
Once installed, you can open the project in your preferred code editor and start modifying it.

To view the changes in real-time, you can use the simulator for your platform (iOS or Android) or connect your mobile device to your computer and run the app on it.
